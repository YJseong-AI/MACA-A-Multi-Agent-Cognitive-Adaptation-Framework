# ResEmoteNet Weights

The pre-trained ResEmoteNet weights are not included in this repository.

Place the weights here:

```
approach/fer2013_model.pth
```

Recommended options to obtain the weights:
- Download from your hosted link (e.g., Release/Drive/Server) and place as above
- Or run:

```bash
bash approach/get_model.sh
```

After download, verify the file is present and readable by Python.
